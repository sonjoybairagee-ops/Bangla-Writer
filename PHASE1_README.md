## 🚀 Quick Start

### 1. Environment Setup (2 minutes)

আপনার `.env.local` file এ add করুন:

```env
# Database (আপনার Neon URL already আছে)
DATABASE_URL="postgresql://neondb_owner:..."

# NextAuth Secret (Generate করুন)
NEXTAUTH_SECRET="run: node -e \"console.log(require('crypto').randomBytes(32).toString('base64'))\""
NEXTAUTH_URL="http://localhost:3000"

# OpenAI (https://platform.openai.com/api-keys থেকে নিন)
OPENAI_API_KEY="sk-proj-..."

# App Config
NEXT_PUBLIC_APP_URL="http://localhost:3000"
```

### 2. Start Server

```bash
npm run dev
```

### 3. Register & Login

```
http://localhost:3000/register
```

### 4. Start Creating!

**Go to: `/dashboard/writer-pro`**

## 🎯 Frameworks Explained

### 1. AIDA (Best for Sales)
```
Attention → Grab with hook
Interest → Build curiosity
Desire → Show benefits
Action → Clear CTA
```

### 2. PAS (Best for Pain Points)
```
Problem → Identify issue
Agitate → Make it urgent
Solution → Your product
```

### 3. BAB (Best for Transformation)
```
Before → Current pain
After → Desired outcome
Bridge → How to get there
```

### 4. Story (Best for Engagement)
```
Hook → Engaging start
Build → Develop narrative
Conflict → Challenge
Resolution → Outcome
Lesson → Takeaway
```

### 5. 4P (Best for Conversion)
```
Promise → Bold claim
Picture → Paint vision
Proof → Evidence
Push → Force action
```
