### 7. Platform Optimization

**Facebook Ads (30s):**
- Hook: ≤3s (MUST grab attention)
- Product reveal: 6-10s
- CTA: Last 5s
- Fast cuts: 1.5s average
- Captions: Required
- Human face: Within 2s

**TikTok (60s):**
- Hook: ≤2s (critical)
- Fast cuts: 0.8s average
- Pattern interrupt: Every 3s
- Big captions: Always on
- Human face: Within 2s
- High energy throughout

**YouTube Shorts (60s):**
- Hook: ≤3s
- Pattern interrupt: Every 4s
- Moderate pace: 1.2s cuts
- Captions: Recommended
- Retention focus

**Instagram Reels (90s):**
- Hook: ≤2s
- Product reveal: 8-15s
- Moderate pace: 1.0s cuts
- Music: Trending sounds
- Captions: Required

## 📊 Technical Implementation

### Files Structure
```
lib/ai-studio/
  └─ director-engine.ts              # Filmmaking rules + AI

app/api/ai-studio/
  ├─ split-scenes/route.ts           # Scene detection API
  └─ direct-scene/route.ts           # Per-scene direction API

components/ai-studio/
  ├─ script-panel.tsx                # Script input
  ├─ visual-director.tsx             # Main AI direction
  ├─ timeline-blueprint.tsx          # Timeline view (NEW)
  ├─ mood-board-panel.tsx            # Style guide (NEW)
  ├─ stock-footage-panel.tsx         # Footage keywords (NEW)
  ├─ checklist-panel.tsx             # Editing tasks (NEW)
  ├─ storyboard-panel.tsx            # Visual cards
  ├─ shot-list-panel.tsx             # Production table
  ├─ motion-plan-panel.tsx           # Animation specs
  └─ export-panel.tsx                # Export options
```

### Key Functions (director-engine.ts)
- `generateStockFootageKeywords()` - Extract searchable keywords
- `generateEditingChecklist()` - Create step-by-step tasks
- `getStyleReference()` - Determine MrBeast vs Apple style
- `generatePremiereProSettings()` - Premiere-specific settings
- `generateCapCutAnimations()` - CapCut animations
- `generateAfterEffectsSettings()` - After Effects settings
- `generateMoodBoard()` - Overall style analysis
- `applyPlatformRules()` - TikTok vs YouTube optimization
